# User Journey Map

Five stages. Each row: what the user feels, what we show, the conversion job, and the viral hook.

## Stage 0 — Arrival (cold open)
| | |
|---|---|
| **Emotion** | Curious, skeptical ("another quiz?") |
| **Screen** | Landing: starfield, one line — *"What do you actually believe?"* — and **one live dilemma already on screen** (e.g. "Would you sacrifice one to save five?"). No signup. |
| **Job** | Convert scroll → first answer in <5s. The first answer is the real conversion event. |
| **Hook** | The machine reacts immediately ("Interesting. The utilitarians are smiling.") → Akinator effect. |

## Stage 1 — The Gauntlet (the quiz)
| | |
|---|---|
| **Emotion** | Engaged, reflective, a little provoked |
| **Screen** | Full-screen dilemma cards, one at a time. Agree↔Disagree slider with intensity, or "It's complicated / I refuse." Progress as a forming constellation, not a boring bar. Ambient "the machine is thinking" line every few questions. |
| **Job** | Maintain momentum to ≥75% completion. Length feels short because each card is a micro-story. 18 core questions (~2.5 min). |
| **Hook** | "You're contradicting yourself…" callouts make it feel intelligent and personal. |

## Stage 2 — The Reveal (the payoff)
| | |
|---|---|
| **Emotion** | Anticipation → delight/surprise |
| **Screen** | Wrapped-style suspense build → constellation snaps into your **primary philosopher** portrait + line ("You think like **Nietzsche**.") → blend bars animate in → **Tribe** badge → **Rarity** ("rarer than 94%"). |
| **Job** | Deliver the identity reward strong enough to trigger a share *right here*. |
| **Hook** | Big primary share CTA + "Was the machine right?" (agree/disagree → more data + social proof). |

## Stage 3 — The Dossier (depth = credibility)
| | |
|---|---|
| **Emotion** | "Okay, this actually gets me" |
| **Screen** | Per-dimension stances, your **core tension** ("You crave meaning but reject the divine — the existentialist's burden"), **shadow philosopher** (the view you're furthest from), signature quote, reading list. Premium teaser blurs the "Deep Report." |
| **Job** | Build trust → fuel sharing and the upgrade path. |
| **Hook** | "Unlock your shadow philosopher by sharing" / "Deep report → upgrade." |

## Stage 4 — The Spread (loops)
| | |
|---|---|
| **Emotion** | Belonging, playfulness, social curiosity |
| **Screen** | Share hub (card formats), **Compare with a friend**, **Battles** ("side with Nietzsche?"), **Chat with your philosopher** (3 free msgs), **Tribe page**, **Daily Dilemma** to come back tomorrow. |
| **Job** | Convert the result into ≥1 outbound action (North Star) and ≥1 return reason. |
| **Hook** | Compare-invite + battle-recruit + daily streak start. |

## Returning user journey
Daily Dilemma push/email → answer → see global split + streak++ → maybe a new weekly Battle → occasional "retake — have you changed?" (results history is premium) → seasonal "Philosophical Year" Wrapped.

## Friction map (where we lose people & the fix)
- **Signup wall too early** → *fix:* quiz is fully anonymous; account only requested to *save/compare/chat* (value-gated, not blocking the result).
- **Quiz too long** → *fix:* 18 core Qs, cinematic pacing, visible "almost there," option to "go deeper (+7)" *after* the result.
- **Generic result** → *fix:* blend + tension + shadow + rarity make it specific.
- **Dead-end after result** → *fix:* Stage 4 always offers a next action.
